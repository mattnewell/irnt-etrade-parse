* Vaguely assumes Python 3.10
* pip install pandas
* pip install pdfquery

I manually downloaded each Release Confirmation PDF from the
E*Trade website. Gotta be careful with that, though.
There's no unique identifier in these files, and it would be easy to either have duplicate or missing records. 

Place downloaded files in a directory called input in the same directory as the script.

Run. Output is creatively named output.csv. Enjoy.
